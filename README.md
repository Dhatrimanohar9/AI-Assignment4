# AI Assignment 4 - CSP Problems

## 📌 Description
This project implements different problems using Constraint Satisfaction Problem (CSP) techniques.

## 🧩 Problems Covered

1. Australia Map Coloring
2. Telangana Map Coloring
3. Sudoku Solver
4. Cryptarithmetic Problem (SEND + MORE = MONEY)

## ⚙️ Approach

All problems are solved using:
- Backtracking Algorithm
- Constraint Checking

## ▶️ How to Run

Run each file using:

```bash
python3 map_coloring_australia.py
python3 telangana_map_coloring.py
python3 sudoku_solver.py
python3 cryptarithmetic.py
